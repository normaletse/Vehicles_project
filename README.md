# Vehicles Project – Aplicación Web con Streamlit

Esta aplicación web fue desarrollada con **Streamlit** y permite realizar un análisis exploratorio
de anuncios de venta de vehículos usados.

## Funcionalidades

- Visualización interactiva de los datos de vehículos.
- Botón para construir un **histograma** del odómetro.
- Botón para construir un **gráfico de dispersión** que muestra la relación entre
  el precio y el año del modelo.
- Gráficos interactivos creados con **Plotly Express**.

## Tecnologías utilizadas

- Python
- Pandas
- Streamlit
- Plotly Express
